# Smart Contracts

## CortexToken (ERC-20)
- Fixed supply: 1,000,000,000 CORTEX
- ERC20Votes for governance
- ERC20Permit for gasless approvals

## CortexVault (ERC-4626)
- Accepts ETH and USDC deposits
- Issues cVault shares
- AI-managed allocation
- Fee structure: 2% management, 20% performance, 0.5% withdrawal

## CortexStaking
- Lock tiers: None (1x), 1mo (1.5x), 3mo (2x), 6mo (2.5x)
- WETH rewards from protocol fees

## CortexGovernor
- OpenZeppelin Governor
- 4% quorum, 3-day voting period
- 24h timelock delay

## Treasury
- 50% to stakers, 50% to treasury
