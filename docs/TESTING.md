# Testing Guide

## Running Tests
```bash
npm test
```

## Test Categories
- Unit tests: `__tests__/lib/`
- Integration: `__tests__/integration/`
- E2E: `__tests__/e2e/`
- Snapshots: `__tests__/snapshots/`
- Security: `__tests__/security/`
- Performance: `__tests__/performance/`
- Accessibility: `__tests__/a11y/`
