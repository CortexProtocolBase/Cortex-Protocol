# Contributing to CORTEX Protocol\n\n## Getting Started\n1. Fork the repo\n2. Clone locally\n3. Install dependencies: `npm install`\n4. Copy `.env.example` to `.env.local`\n5. Run dev server: `npm run dev`\n\n## Pull Requests\n- Create a feature branch from `main`\n- Write tests for new features\n- Ensure `npm run build` passes\n- Submit PR with clear description\n\n## Code Style\n- TypeScript strict mode\n- Tailwind CSS for styling\n- Functional components with hooks\n- Named exports preferred
